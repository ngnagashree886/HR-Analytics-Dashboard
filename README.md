# HR-Analytics-Dashboard
Interactive HR Analytics Dashboard built using Power BI, DAX, and Power Query to analyze employee attrition, workforce demographics, salary trends, and department-wise performance.
# HR Analytics Dashboard

## Project Overview
This project analyzes employee attrition, workforce demographics, education levels, and salary distribution using Power BI.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary
- Average Age

## Key Insights
- IT department has the highest attrition.
- Employees aged 26–35 form the largest workforce segment.
- Bachelor's degree holders represent the largest employee group.
